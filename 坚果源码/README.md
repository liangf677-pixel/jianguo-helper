# 坚果 - 个人本地全能任务管理与生活助手

## 项目简介

一个**纯本地运行、无需联网、不收集任何数据**的个人专属任务管理与生活助手桌面应用（.exe格式）。

专为个人日常使用设计，帮助用户高效管理任务、养成良好习惯、避免遗忘重要事项，同时集成时钟、闹钟、天气等实用生活功能。

## 技术栈

- **核心框架**: Electron 28.x
- **前端技术**: 纯 HTML5 + CSS3 + Vanilla JavaScript
- **数据存储**: electron-store (本地加密存储)
- **打包工具**: electron-builder (生成 .exe 文件)

## 功能模块

### 任务管理系统
- 快速创建/编辑/复制/删除任务
- 多种视图：今日、即将到期、全部、已完成、已逾期
- 按分类、优先级筛选和搜索
- 批量操作：批量完成、批量删除
- 拖拽排序
- 桌面通知提醒
- 重复任务支持（每天/每周/每月/每年）

### 时钟与闹钟
- 实时数字时钟（支持12/24小时制）
- 多闹钟管理，支持重复日期
- 全屏置顶闹钟提醒
- 稍后提醒功能
- 倒计时器（含番茄钟预设）
- 秒表（计次功能）

### 天气预报
- 集成高德地图天气 API
- 当前天气详情（温度/湿度/风向/风力）
- 未来3天预报
- 支持多城市切换
- 自动定时更新

### 便签
- 多颜色分类便签
- 自动保存内容
- 自由创建和删除

### 纪念日/生日
- 添加重要日期
- 倒计时显示
- 提前提醒设置

### 桌面专属功能
- 系统托盘最小化
- 开机自启
- 全局快捷键
- 窗口大小/位置记忆
- 数据导出/导入备份

## 环境要求

- Windows 10 或更高版本
- Node.js 18.x 或更高版本（开发/打包时需要）
- npm 9.x 或更高版本

## 快速开始

### 1. 安装 Node.js

如果还没有安装 Node.js，请前往 [https://nodejs.org/](https://nodejs.org/) 下载并安装 LTS 版本。

安装完成后，打开命令提示符（CMD）或 PowerShell，验证安装：

```bash
node --version
npm --version
```

### 2. 安装项目依赖

在项目根目录下运行：

```bash
cd task-assistant
npm install
```

### 3. 运行应用（开发模式）

```bash
npm start
```

应用窗口将自动打开。

### 4. 打包为 EXE 文件

```bash
npm run build
```

打包完成后，可执行文件将生成在 `build/` 目录下：
- `任务管理助手-1.0.0-portable.exe` - 单文件便携版

直接双击即可运行，无需安装。

### 5. 生成安装包

```bash
npm run build:installer
```

## 天气功能配置

应用已预置高德地图 API 密钥，开箱即用。如需更换：

1. 前往 [高德开放平台](https://lbs.amap.com/) 注册账号
2. 创建应用并获取 Web服务 API Key
3. 在应用【设置 > 天气设置】中填入新的 API Key

## 数据存储

所有数据存储在用户本地电脑的应用数据目录中（使用 electron-store），不会发送到任何服务器。

### 数据备份与恢复

- **导出**: 设置 → 数据管理 → 导出备份 → 选择保存位置
- **导入**: 设置 → 数据管理 → 导入数据 → 选择备份文件

建议定期导出数据作为备份。

## 隐私说明

- 所有数据100%存储在本地电脑
- 不收集任何用户信息
- 不跟踪用户行为
- 不使用任何分析工具
- 除天气功能外，所有功能完全离线可用
- 天气功能可随时禁用
